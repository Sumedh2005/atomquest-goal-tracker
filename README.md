## Features

### Admin
- Organization dashboard and analytics
- Cycle configuration management
- Shared goal management
- Achievement reports and audit trails
- Organization-wide performance monitoring

### Manager
- Team performance dashboard
- Goal and check-in review system
- Approval queue management
- Team progress tracking

### Employee
- Personal dashboard
- Goal sheet creation and management
- Quarterly check-ins
- Goal tracking and progress updates
